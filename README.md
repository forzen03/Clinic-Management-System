# Clinic System (JDeveloper Course Project)

This project is a sample clinic management system developed as part of a course at Al Zaytoonah University. It demonstrates the use of Oracle JDeveloper for building enterprise applications with database integration, Java, and Oracle ADF.

## Project Structure
- **DatabaseModel/**: Contains database schema, offline database files, and related configuration.
- **Model/**: Java classes, entity objects (EO), view objects (VO), and application module for business logic and data access.
- **UML/**: UML diagrams and use case scenarios for system design and documentation.
- **ViewController/**: JSF pages and ADF controller/view logic for the web interface.

## Technologies Used
- Oracle JDeveloper
- Oracle ADF (Application Development Framework)
- Java
- JSF (JavaServer Faces)
- UML for design

## How to Open the Project
1. Open Oracle JDeveloper.
2. Use `File > Open...` and select the `Clinic_System.jws` workspace file.
3. Build and run the application from within JDeveloper.

## Features
- Manage doctors, patients, and appointments
- Search and view records
- UML diagrams for system design

## Notes
- This project is for educational purposes and may require Oracle JDeveloper and a compatible database setup.
- Database files are included for offline use; adjust settings as needed for your environment.
